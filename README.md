# 📖 Sobre o Projeto

O **WMT (Work Meets Talent)** é um aplicativo mobile criado para conectar jogadores de basquete, facilitar a organização de partidas e incentivar o desenvolvimento esportivo através de estatísticas individuais e avaliações das quadras.

O projeto foi desenvolvido como parte da disciplina de Desenvolvimento Web App do curso de **Análise e Desenvolvimento de Sistemas**, tendo como objetivo aplicar conceitos de desenvolvimento mobile, APIs REST, integração entre frontend e backend e persistência de dados.

---

# 🎯 Objetivos

O WMT busca oferecer uma plataforma completa para a comunidade do basquete, permitindo que qualquer jogador possa:

- 🏀 Organizar partidas;
- 🤝 Encontrar outros jogadores;
- 📍 Descobrir e avaliar quadras;
- 📊 Acompanhar sua evolução;
- ⭐ Compartilhar experiências com a comunidade.

---

# ✨ Funcionalidades

## 👤 Usuário

- Login utilizando nome de usuário;
- Perfil do jogador;
- Encerramento de sessão.

---

## 🏀 Jogos

- Criar partidas;
- Participar de partidas;
- Exclusão individual de partidas;
- Exclusão de todas as partidas.

---

## 📍 Quadras

- Cadastro de novas quadras;
- Avaliação utilizando sistema de estrelas;
- Ranking automático das melhores quadras;
- Média das avaliações.

---

## 📊 Estatísticas

- Registro de partidas;
- Dashboard de desempenho;
- Média de pontos;
- Média de rebotes;
- Média de assistências;
- Melhor partida;
- Histórico de partidas;
- Exclusão de partidas.

---

## 👤 Perfil

- Nome do jogador;
- Jogos disputados;
- Média de pontos;
- Melhor partida;
- Quadra favorita;
- Nota média da quadra favorita;
- Insígnia dinâmica do jogador.
