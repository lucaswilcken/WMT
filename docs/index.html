📖 Sobre o Projeto

O WMT é um aplicativo mobile desenvolvido em React Native com o objetivo de conectar jogadores de basquete, facilitando a organização de partidas, avaliação de quadras e acompanhamento do desempenho individual.

✨ Funcionalidades

  • ✅ Login com nome de usuário
  • 🏀 Criação de partidas
  • 🤝 Participação em partidas
  • 📍 Cadastro de quadras
  • ⭐ Avaliação por estrelas
  • 🏆 Ranking das quadras
  • 📊 Estatísticas do jogador
  • 👤 Perfil integrado
  • 🚪 Encerramento de sessão
